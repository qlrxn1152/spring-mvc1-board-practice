Spring MVC1에서 배운 요청 매핑, 컨트롤러, 모델, 뷰, 폼 처리, 리다이렉트 흐름을 직접 연습하기 위한 게시판 프로젝트입니다.

게시글 목록 보기 게시글 작성하기
연습 중인 MVC 흐름

Controller

@GetMapping, @PostMapping으로 요청을 처리합니다.

Model

Controller에서 조회한 데이터를 View로 전달합니다.

View

Thymeleaf 템플릿으로 게시글 목록, 상세, 작성, 수정 화면을 렌더링합니다.

Redirect

저장, 수정, 삭제 후 PRG 패턴으로 다시 요청을 보냅니다.
